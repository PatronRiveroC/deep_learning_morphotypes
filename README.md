## 📖 Abstract
The delimitation of morphotypes in cryptic lineages exposes the fundamental limitations of classical linear statistics and subjective human perception, which often fail to capture complex, non-linear phenotypic variations. To address this, we introduce a scalable, unsupervised Machine Learning framework that couples Graph Autoencoders (GAE) with community detection algorithms to map high-dimensional morphospaces. Unlike traditional models that treat biological specimens as isolated data points, our approach represents morphological multidimensional data as an interconnected network, utilizing a k-nearest neighbors’ graph to learn topological relationships and compress them into an optimized latent space. As an empirical case study, we applied this pipeline to 484 specimens of the morphologically conservative Neotropical pitviper genus Porthidium, using 21 linear and pholidosis (scale counts) traits. The GAE successfully decoupled morphological structure from traditional taxonomic boundaries, identifying 12 distinct morphotypes with high structural modularity (Q = 0.6973) but low taxonomic concordance (NMI = 0.2812). Ultimately, this AI-driven framework provides an objective and highly scalable solution for deep phenotyping. Capable of handling massive biological datasets without prior taxonomic bias, this methodology can be universally applied to other taxa and especially valuable in cryptic lineages, which morphological boundaries become abstracts. Ultimately, serve as a robust diagnostic step for integrative taxonomy, systematics and potentially applied to evolutionary biology.
